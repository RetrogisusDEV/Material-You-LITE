# Material You Lite
**Plis Help to Translate to english**

Material You Lite es una plantilla para introducir rapida y comodamente Material Desing 3 en cualquier app

En android Studio o en Reverse(Smali/Java)

## Compartibilidad

- App Theme
- Linear Layout
- Relative Layout
- Frame Layout

- TextView
- ImageView
- View
- Button
- CheckBox
- SeekBar
- RadioButton

- ScrollView and Horizontal ScrollView

- Dialog
